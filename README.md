# AgriCam - Interface Utilisateur (Frontend)

Bienvenue sur le dépôt frontend d'**AgriCam**, un projet de fin d'études conçu pour moderniser et optimiser la chaîne de valeur agricole. 

Ce dépôt contient le code source de l'interface utilisateur. Il gère la navigation, l'affichage des tableaux de bord et l'expérience visuelle globale de la plateforme avant son intégration avec le serveur backend.
